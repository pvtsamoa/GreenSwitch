## The Truth About Cannabis

**Cannabis sativa L.** is a plant species with incredible industrial potential.

There are two main varieties:
1. **High-THC Cannabis** (>0.3% THC) - Recreational/medical use
2. **Low-THC Cannabis** (<0.3% THC) - Industrial/material use

**"Hemp" is simply the common name for low-THC industrial cannabis.**

It's the same plant. The same species. Just bred for different purposes.
