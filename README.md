# GreenSwitch

**Same need. Better result.** 🌿

Telegram bot prototype that compares everyday petroleum-derived products to sustainable **industrial cannabis** (low-THC hemp) alternatives. Educational only — non-psychoactive, legal industrial applications (bioplastics, fabrics, insulation, etc.). No recreational/medical cannabis.

## Live Demo
Telegram bot: (@GreenSwitch_bot)(t.me/GreenSwitch_bot)
Try: `/search plastic bottle` or `/search polyester clothing`

## Features
- `/search` command for side-by-side comparisons
- Modular Node.js structure with handlers, services (AI, cache, database, rate limiting)
- Data in `src/data/comparisons.js`

## Tech Stack
- Node.js
- Telegram Bot API
- Planned: Next.js web frontend, SQLite/Supabase, Claude AI fallback for dynamic searches

## Why this project?
Veteran-built (U.S. Army UH-60 helicopter repair, 5+ years). Applying systems troubleshooting and discipline to accelerate sustainable alternatives to petroleum.

## Setup (local)
1. `npm install`
2. Copy `.env.example` to `.env` and add your BOT_TOKEN
3. `node index.js`
